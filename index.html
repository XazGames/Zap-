История обновлений

v1.0.3 (Текущая)

  • Полный редизайн: строгий стиль, адаптивность под мобилки, SVG иконки.
  • Продвинутая Тема: Смена светлой и темной темы (кнопка сверху справа) с сохранением.
  • Система Окон: Игры теперь открываются в окнах, которые можно растягивать! Движок подстраивается под размер.
  • Discord Parody: Добавлены фейковые звонки (аудио, видео, экран) из чата.
  • Контекстное меню: Нажми правой кнопкой мыши по сообщению в чате для удаления или жалобы.
  • Умные уведомления: Никаких системных alert, только красивые Toasts.
  • Умные профили: Клик по аватарке пользователя перекидывает в его профиль.
